# 🎬 Top 100 Movies – Python Web Scraping Project

A Python script that scrapes the **Top 100 Movies of All Time** from [IMDb](https://www.imdb.com/) and saves them into a text file. Perfect beginner project for learning **web scraping** using `BeautifulSoup`.

---

## 🔍 What It Does

- Connects to the IMDb Top 100 movies list
- Extracts movie titles and ranks using HTML parsing
- Saves the list in a clean format to a `.txt` file

---

## 🛠️ Built With

- 🐍 **Python 3**
- 🌐 `requests` – to fetch webpage content
- 🧼 `BeautifulSoup (bs4)` – to parse and extract data
- 📄 `with open()` – to write the scraped data to a file

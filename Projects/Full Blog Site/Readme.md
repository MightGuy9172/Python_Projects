# Full Blog Site

A comprehensive blog platform built with Python, allowing users to create, edit, and manage blog posts with authentication and commenting features.

## Features

- User registration and login
- Create, edit, and delete blog posts
- Commenting system for posts
- Responsive design for desktop and mobile
- Admin dashboard for managing users and posts

## Technologies Used

- Python (Flask/Django)
- HTML, CSS, JavaScript
- SQLite/PostgreSQL (database)
- Bootstrap (UI framework)

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python app.py
   ```
4. Access the site at `http://localhost:5000`

## Configuration

- Update database settings in the configuration file as needed.
- Set environment variables for secret keys and other sensitive data.

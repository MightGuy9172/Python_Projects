# 🐢 Turtle Race – Fun with Python Turtle Graphics

A colorful and entertaining **turtle race simulator** built using Python’s `turtle` module. Choose your favorite turtle and watch them race across the screen!

---

## 🎮 Game Description

The Turtle Race is a graphical game where six colorful turtles line up at the starting line. You pick a color to bet on, and then the race begins. The turtles move forward randomly, and the first one to reach the finish line wins!

---

## 💡 Features

- Interactive terminal input to place your bet
- Colorful turtle graphics
- Randomized race results for replayability
- Beginner-friendly Python code

---

## 🛠️ Technologies Used

- 🐍 Python 3
- 🐢 Turtle module (standard Python library)
